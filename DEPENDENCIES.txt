TRIALOGUE - DEPENDENCIES & INSTALLATION
========================================

REQUIRED DEPENDENCIES
=====================

Python Version:
  • Python 3.8 or higher

Python Packages:
  • flask
  • langchain-openai
  • langchain-core
  • openai

INSTALLATION COMMAND
====================

Install all dependencies:

pip install flask langchain-openai langchain-core openai

OR

pip3 install flask langchain-openai langchain-core openai

REQUIREMENTS.TXT FILE
=====================

Install with:

pip install -r requirements.txt

OPENAI API KEY
==============

You also need an OpenAI API key:

1. Sign up at: https://platform.openai.com/api-keys
2. Create a new API key
3. Set it as an environment variable:

   macOS/Linux:
   export OPENAI_API_KEY='your-key-here'

   Windows (Command Prompt):
   set OPENAI_API_KEY=your-key-here

   Windows (PowerShell):
   $env:OPENAI_API_KEY='your-key-here'

==========

Once dependencies are installed and API key is set, run:

python server.py

Then open browser to: http://127.0.0.1:5000
